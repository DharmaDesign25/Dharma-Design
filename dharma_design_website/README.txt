# Dharma Design Website

## How to Use

1. Upload the contents of this folder to any static web hosting platform:
   - [GitHub Pages](https://pages.github.com/)
   - [Netlify](https://www.netlify.com/)
   - [Vercel](https://vercel.com/)
   - Any traditional web hosting service

2. Access your website through the generated URL.

3. Add your product images in the "images" folder and link them in the gallery section of `index.html`.
